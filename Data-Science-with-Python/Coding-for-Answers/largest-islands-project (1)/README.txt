🏝️ World's Largest Islands Analysis
This project explores a dataset of the world's largest islands using Python and pandas. It demonstrates core data analysis skills including data cleaning, sorting, grouping, filtering, and visualization with matplotlib.

📂 Repository Structure
largest-islands-project/
├── largest-islands-Analysis.ipynb   # Main notebook with structured analysis
├── largest-islands.csv              # Dataset used for analysis
└── README.md                        # Project overview and instructions

📊 Project Objectives
The notebook answers the following questions:
•	What are the 10 largest islands in the tropics?
•	Which islands are the largest in each region?
•	What does the relationship between island rank and area look like?
•	Which islands are shared by multiple countries?

🧪 Tools & Technologies
•	Python 3.13
•	pandas for data manipulation
•	matplotlib for visualization
•	Jupyter Notebook (developed in Visual Studio Code)

📈 Key Insights
•	The largest tropical islands include New Guinea and Borneo.
•	Europe and Asia host multiple large islands shared by more than one country.
•	The area vs. rank graph shows a steep drop-off after the top few islands.

📁 Dataset
The dataset `largest-islands.csv` contains columns such as:
•	`island`: Name of the island
•	`area`: Surface area in square kilometers
•	`rank`: Global rank by area
•	`region`: Geographic region
•	`countries`: Countries sharing the island
•	`climate`: Climate zone (e.g., tropical, temperate)

🚀 How to Run
1.	Clone the repository:
   git clone https://github.com/your-username/largest-islands-project.git
2.	Open `largest-islands-Analysis.ipynb` in VS Code or Jupyter Notebook.
3.	Ensure `largest-islands.csv` is in the same directory.
4.	Run all cells to reproduce the analysis.
